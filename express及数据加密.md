# express

## Express

- express是一个基于Node.js平台的极简、灵活的web应用平台框架(模块)

## 数据加密算法

### md5

```js
yarn add md5
npm install md5



const md5 = require('md5');
let password = '123456789';
console.log(md5(password));
```

### sha1

```js
yarn add sha1
npm install sha1



const sha1 = require('sha1');
let password = '123456789';
console.log(sha1(password));
```
